# Review Prep Checklist

## Repo and project hygiene
- [ ] README is current
- [ ] GitHub Project board is up to date
- [ ] issues are labeled and organized by week
- [ ] pull requests include screenshots or demo links
- [ ] evidence log is updated

## Project proof
- [ ] Project 1 deployed and working
- [ ] Project 2 deployed and working
- [ ] Capstone demo path is stable
- [ ] mobile layout checked
- [ ] no obvious console errors
- [ ] form labels and focus states checked

## Demo assets
- [ ] 3-5 minute walkthrough script prepared
- [ ] screenshots saved
- [ ] short demo video recorded or planned
- [ ] README explains problem, stack, features, and lessons learned
- [ ] can explain what Bootstrap handled vs what custom CSS and JavaScript handled

## Reflection
- [ ] list 3 strengths from the sprint
- [ ] list 3 improvement areas
- [ ] list next 30 days of growth after the review
